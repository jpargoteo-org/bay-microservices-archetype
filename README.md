# bay-microservices-archetype
The repository for the bay microservices archetype project.

## Archetype execution
Execute the following command into the terminal:

```sh
## Windows
$ mvn archetype:generate "-DarchetypeGroupId=es.jpargoteo.bay" "-DarchetypeArtifactId=bay-microservice-archetype" "-DarchetypeVersion=0.0.1-SNAPSHOT"

## Linux
$ mvn archetype:generate -DarchetypeGroupId=es.jpargoteo.bay -DarchetypeArtifactId=bay-microservice-archetype -DarchetypeVersion=0.0.1-SNAPSHOT
```
